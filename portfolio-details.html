Detalhes e Informações

Este foi um projeto desenvolvido em grupo pela matéria de IHC/UX ministrada pela profa. Kamila Rios durante o segundo semestre de 2023. Este projeto é um e-commerce de PetShop, feito com um foco ao cliente e a uma maior usabilidade, gerando maior satisfação para o cliente. Alguns diferenciais dos demais PetShop's são; Função de Libras; Categorias bem definidas; Aniversariantes do mês, entre outras funcionalidades.
